# Handwritten-Text-Recognition-HTR-System
An optimized Handwritten Text Recognition (HTR) system based on CRNN (PyTorch). Features ONNX/INT8 quantization for 2x CPU inference speed, OCR-aware spell checking, and a real-time Flask Web App. Achieves 1.7% CER.
